06 · Search Quality

Recent Improvements — Inspired by Onyx OSS

α

Smarter Score Fusion

BM25 keyword + vector scores are normalized then blended with a tunable weight. α=0.7 favors semantic retrieval.

Replaces: Reciprocal Rank Fusion
↑ precision for semantic queries
λ

Time-Decay Scoring

Older documents automatically rank lower using exponential decay on updated_at.

~50% penalty after 231 days
Stale docs stay below fresh ones
Σ

Token Budget Assembly

Context greedily fills a 6,000-token budget by reranker score instead of a hard relevance cutoff.

Replaces: fixed threshold 0.3
Adapts to actual content length
